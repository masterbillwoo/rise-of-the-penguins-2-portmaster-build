Rise of the Penguins 2 - PortMaster Port
==========================================

Game Information:
-----------------
Developer: Bill Woo
Publisher: Pwoo
Genre: Platformer / Action / Adventure
Players: 1
Runtime: Godot 4.5

Installation:
-------------
Place the game's .pck file in the riseofthepenguins2/gamedata/ folder as riseofthepenguins2.pck

[itch.io page](https://billwoo.itch.io/rise-of-the-penguins-2-penguins-dark-ascent)
[steam page](https://store.steampowered.com/app/3597710/Rise_of_the_Penguins_2_Penguins_Dark_Ascent/)

Controls:
---------
D-Pad/Left Stick: Move
A Button: Dive
B Button: Jump
X Button: Dash
Y Button: Attack
L/R Buttons: Use Item
Start: Pause Menu
Select: Inventory
