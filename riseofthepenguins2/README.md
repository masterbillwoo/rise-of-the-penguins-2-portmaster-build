## Controls

| Button         | Action     |
| -------------- | ---------- |
| D-Pad/L-Stick  | Move       |
| A              | Dive       |
| B              | Jump       |
| X              | Dash       |
| Y              | Attack     |
| L/R            | Use Item   |
| START          | Pause Menu |
| SELECT         | Inventory  |
